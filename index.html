About

An implementation of a URL shortening service that stores the value in an array and uses that position in the array as a key.


The Format:

http://shorten.kerkstra.me/a/unique_identifier

The position of the key gets encoded to base 36 (i.e. 0-9, a=10, b=11 ... z = 35) and becomes the unique identifier in the URL.
